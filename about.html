Purpose

The main purpose of this site is to give more in depth descriptions of projects and work experience relevant to my career. A resume can't contain everything, and a LinkedIn can quickly become bloated with videos, images, links, and long descriptions.
Besides that, I'd like a place to share things I've done and things I'm interested in that wouldn't be appropriate for a resume or LinkedIn, somewhat inspired by Richard Stallman's Personal Site .

Why does it look like this?

I wanted a barebones website that that is easy to read and navigate.
I feel that this design conveys information efficiently with minimal clicks.

For a more impressive example of my web design skills, see ixion .

Technical details

This website is a static GitHub page written in HTML and styled with TailwindCSS.
